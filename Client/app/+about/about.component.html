ASP.NET Core

ASP.NET Core is a new open-source and cross-platform framework for building modern cloud based applications.

Business layer developed with ASP.NET Core e.g. Web API, authentication, authorization ect.

Webpack 2

Webpack is a module bundler. webpack takes modules with dependencies and generates static assets representing those modules.

Webpack 2 used to build the Client app e.g. minifying, concatenating ect.

Angular 2

Angular is not an MVC framework, but rather a component-based framework. In Angular an application is a tree of loosely coupled components.

Angular 2 (^v4.0) has been used for the Client side of the website using Typescript.

Material Design

Material Design (codenamed Quantum Paper) is a design language and styling guide developed in 2014 by Google.

Client side is styled using Angular Materials 2 and Material Design principles have been followed.
